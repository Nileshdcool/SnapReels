## Run Locally

Clone the project

```bash
  git clone https://github.com/Nileshdcool/SnapReels.git
```

Go to the project directory

```bash
  cd SnapReels
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
